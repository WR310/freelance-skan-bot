## 📌 О проекте
**IsmailScannerBot** — это Telegram-бот для автоматизированного сбора, фильтрации и маршрутизации целевых тендерных заявок. 

Проект решает задачу рутинного поиска заказов в B2B-секторе (включая строительную отрасль). Бот непрерывно мониторит заданные торговые площадки, отсеивает мусор по заданным критериям и присылает готовые лиды прямо в мессенджер, экономя часы работы тендерного специалиста.

## ⚙️ Ключевой функционал
- **Сбор данных в реальном времени:** Парсинг целевых площадок (настраивается под нужды заказчика).
- **Многоуровневая фильтрация:** Отбор тендеров по ключевым словам, минус-словам, региону и сумме (НМЦК).
- **Анти-дубликат:** Использование базы данных для отслеживания уже отправленных тендеров.
- **Удобный интерфейс в Telegram:** Карточки тендеров с кратким описанием, сроками и прямыми ссылками на первоисточник.
